Assignment 1b Confusion matrix¶

Through the diagnosis test I predicted 100 report as COVID positive, but only 45 of those were actually positive. Total 50 people in my sample were actually COVID positive. I have total 500 samples. Create confusion matrix based on above data and find

  1. Accuracy
  2. Precision
  3. Recall
  4. F-1 score

Generating Confusion matrix¶

In [ ]:
tp = 45
tn = 395
fp = 55
fn = 5
conf_mat = np.matrix([[tp, fp],[fn, tn]])
print(conf_mat)

Accuracy¶

Accuracy measures how often the model is correct

(TP + TN) / Total Predictions

In [ ]:
#using calculation
accuracy = (tp+tn) / (tp+tn+fn+fp)
accuracy
Out[ ]:
0.88

Precision¶

Of the positives predicted, what percentage is truly positive?

How to Calculate:

TP / (TP + FP)

In [ ]:
precision = tp/(tp+fp)
precision
Out[ ]:
0.45

Recall¶

Of all the positive cases, what percentage are predicted positive?

Sensitivity (sometimes called Recall) measures how good the model is at predicting positives.

This means it looks at true positives and false negatives (which are positives that have been incorrectly predicted as negative).

How to Calculate TP / (TP + FN)

In [ ]:
recall = tp/(tp+fn)
recall
Out[ ]:
0.9

F-1 score¶

F1 score is the harmonic mean of precision and recall and is a better measure than accuracy.

2 ( (precisionrecall) / (precision+recall) )

In [ ]:
f1_score = 2 * ((precision*recall)/ (precision+recall))
f1_score
Out[ ]:
